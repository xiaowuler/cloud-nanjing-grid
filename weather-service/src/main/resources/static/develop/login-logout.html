参数代码

中文名

参数类型

赋值格式/示例

必选/可选

userName

字符串

示例：zhangsan

必选

secret

用户密文

字符串

将用户登录名和用户密码连接起来，进行MD5加密，加密密文作为该参数的值()

必选

{ "retCode" : retCode, "retMsg" : retMsg, "userName" : userName,
