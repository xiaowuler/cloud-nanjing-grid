1、根据开始时间、结束时间查询指定地区指定要素的实况数据

（1）接口调用请求说明

接口ID http://ip:port/ realSearch/findRealNJGridsByArea
参数代码

 中文名

 参数类型

 赋值格式 必选/可选
token 唯一凭证 字符串 登录者访问本系统的接口服务的唯一凭证，由登录时候获得 必选
startRealTime 开始时间 字符串 时间格式为yyyyMMddHHmmss，如20150816090703 必选
endRealTime 结束时间 字符串 时间格式为yyyyMMddHHmmss，如20150816090703 必选
areaCode 地区代码 字符串 取值参见【地区编码】字典定义 必选
elementCode 要素 字符串 取值字典参见【预报要素字典定义】 可选

 

（2）接口响应说明

 

返回值字段说明：

字段名 数据类型 取值说明
retCode Integer 返回码。参见【接口返回码定义】
retMsg

String

 返回码描述：查询成功或查询失败原因
data 数据集合
字段名 数据类型 取值说明
elementCode String 实况数据模式
reals List<Real> 实况信息集合

 

Real
字段名 数据类型 取值说明
realTime String 实况时间：yyyy-MM-ddHH:mm:ss
locations List<Location> 位置信息集合

 

Location
字段名 数据类型 取值说明
loc double[] 集合索引0为经度，1为纬度
gridCode String 网格编码
areaCode String 地区编码
areaName String 地区名称
value Double 实况数据值
